# Getting Started with Onlook

Onlook is available as a web application that lets you visually edit your React and TailwindCSS projects.

## Requirements

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A GitHub account to sign in
- Internet connection

## Accessing Onlook

1. Visit [Onlook web app](https://onlook.com)
2. Sign in with your GitHub account
3. Create a new project or open an existing one

## First Steps

Once you've signed in to Onlook, you can:
- Create a new project
- Import an existing project from GitHub
- Explore sample projects

## Next Steps

Now that you're set up with Onlook, you're ready to [create your first project](/docs/tutorials/first-project).
