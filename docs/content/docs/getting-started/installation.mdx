# Installing Onlook

Onlook is available as a desktop application for macOS (Apple Silicon) and Windows.

## System Requirements

- **macOS**: macOS 11 or later (Apple Silicon)
- **Windows**: Windows 10 or later
- At least 4GB of RAM
- 500MB of available disk space

## Download and Install

1. Visit the [Onlook download page](https://onlook.com/download)
2. Download the appropriate version for your operating system
3. Open the downloaded file
4. Follow the installation instructions for your platform
